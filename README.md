# #01 Installation

### Configure environtment

- runtime `node.js`
- package manager `npm`
- Next.js version `15.3.2`

### Create Next.js App

run:

```bash
npx create-next-app@latest
```

configuration:

- TypeScript `Yes`
- ESLint `Yes`
- Tailwind CSS `Yes`
- `src/` directory `Yes`
- App Router `Yes`
- Turbopack `No`
- Import alias `No`
